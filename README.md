# 📊 Titanic EDA (Exploratory Data Analysis)

## 📌 Objective

The objective of this project is to perform Exploratory Data Analysis (EDA) on the Titanic dataset to understand patterns, relationships, and key insights from the data.

---

## 📂 Dataset

The dataset contains information about passengers such as:

* Age
* Gender
* Passenger Class (Pclass)
* Fare
* Survival Status

---

## ⚙️ Steps Performed

1. Imported necessary libraries (Pandas, Matplotlib, Seaborn)
2. Loaded the dataset using Pandas
3. Explored the dataset using:

   * `head()`
   * `info()`
   * `describe()`
4. Checked for missing values
5. Visualized data using graphs
6. Analyzed relationships using correlation

---

## 📊 Visualizations

* 📈 Age Distribution (Histogram)
* 📦 Fare Distribution (Boxplot)
* 👥 Survival Count
* 🚻 Survival by Gender
* 🔥 Correlation Heatmap
* 🔗 Pairplot

---

## 🔍 Key Insights

* Female passengers had a higher survival rate than males
* Passengers who paid higher fares had better chances of survival
* Younger passengers showed slightly higher survival trends
* Strong relationships exist between some features like Fare and Passenger Class
