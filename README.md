📊 Exploratory Data Analysis (EDA) - Titanic Dataset
📌 Objective

The goal of this project is to perform Exploratory Data Analysis (EDA) on the Titanic dataset to understand the data, identify patterns, and extract meaningful insights.

📂 Dataset

The dataset contains information about passengers such as:

Age
Gender
Fare
Passenger Class
Survival Status
⚙️ Steps Performed
Data loading using Pandas
Data inspection (head(), info(), describe())
Handling missing values
Data visualization using Matplotlib & Seaborn
Correlation analysis
📊 Visualizations
Age Distribution (Histogram)
Fare Distribution (Boxplot)
Survival Count
Survival by Gender
Correlation Heatmap
Pairplot
🔍 Key Insights
Female passengers had a higher survival rate than males
Passengers with higher fares had better survival chances
Younger passengers showed slightly higher survival trends
Some features show strong correlation (e.g., Fare and Pclass)
🛠️ Tools Used
Python
Pandas
Matplotlib
Seaborn
🚀 Conclusion

EDA helps in understanding the dataset and discovering patterns, which is useful before applying machine learning models.
